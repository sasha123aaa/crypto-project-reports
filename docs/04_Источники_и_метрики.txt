ИСТОЧНИКИ И МЕТРИКИ

1. Tokenomist
Использовать для:
— структуры токеномики
— аллокаций
— vesting
— cliff
— календаря разлокировок

2. TokenUnlocks
Использовать для:
— ближайших разлокировок
— monthly unlocks
— графика разлокировок
— давления предложения

3. CryptoRank
Использовать для:
— supply
— FDV
— market cap
— инвесторов
— фондрайзинга
— раундов

4. SoSoValue
Использовать для:
— агрегированных рыночных метрик
— supply-метрик
— narrative dashboards
— активности по секторам

5. DefiLlama
Использовать для:
— TVL
— fees
— revenue
— DEX volume
— data по протоколам и сегментам
— inflows / outflows если доступны

6. Token Terminal
Использовать для:
— revenue
— fees
— users
— мультипликаторов
— роста / снижения экономики проекта

7. Dune
Использовать для:
— кастомных дашбордов
— users
— wallets
— onchain-активности
— whale-tracking
— breakdown по направлениям

8. Arkham
Использовать для:
— поведения крупных кошельков
— фондов
— treasury
— крупных переводов
— концентрации предложения

9. CoinGecko / CoinMarketCap
Использовать для:
— цены
— market cap
— volume
— базовых рыночных метрик

Правило:
— если одна и та же метрика отличается в разных источниках, это нужно указывать
— если данные устарели или спорные, писать об этом прямо
— не подменять фундаментал одним только ценовым графиком
